# ClassmateJS Agent Notes

- Test driven development is preferred, but not required. If you add or change code, please add or change tests to match. If you work on new features without tests, explain to the prompter why this is needed.
- This repo is `classmatejs`, a TypeScript npm workspace with React and Solid adapters.
- ClassmateJS creates class-name-focused components with template literals, dynamic props, variants, extension helpers, inline style support, and `tailwind-merge` class merging.
- Prefer arrow functions when adding or changing TypeScript or JavaScript code, matching the current source style.
- Always run `npm run verify` after making changes, including prompt or instruction changes such as edits to this file, before handing work back.
